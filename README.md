# PHP-Docker
Docker images for PHP based on the official images.
Based on the Wordpress docker image, but without Wordpress.

## PHP extions added:
* GD
* Mysqli
* Opcache 
* Zip

## Apache modifications:
* Enable mod_rewrite
* Enable SSL (default self-signed SSL)

## Versions available
* PHP7.4 - Apache
* PHP7.3 - Apache
* PHP7.2 - Apache
* PHP5.6 - Apache (outdated, and unsupported)
